# Sistema de Gestión de Convivencia Escolar

> Proyecto semestral — Ingeniería de Software 2  
> Universidad de Concepción — Equipo 02  
> Integrantes: Diego Alday, Ignacio Jara, Gabriela Muñoz, Bastián Zapata

## Descripción

Sistema web para el registro, seguimiento y análisis de incidentes de convivencia escolar en establecimientos educacionales. Reemplaza los mecanismos manuales (anotaciones en papel, correos, carpetas físicas) por una plataforma digital integrada y multi-rol.

## Estructura del proyecto

```
convivencia-escolar/
├── backend/        # API REST con Node.js + Prisma + PostgreSQL
├── frontend/       # Interfaz web (por definir)
├── docs/           # Informes, diagramas y documentación
└── README.md
```

## Tecnologías

| Capa | Tecnología |
|---|---|
| Backend | Node.js, Prisma ORM |
| Base de datos | PostgreSQL |
| Frontend | Por definir |

## Requisitos previos

- Node.js 18+
- PostgreSQL 14+
